Gravity Penguin

My first ever big game project, and the first in the Gravity Penguin series. I came up with the character design and gameplay premise, which is based on the 2D Sonic the Hedgehog games for the SEGA Genesis, and then pitched that to my class. It was the highest voted game to work on, so I and a small team of 5 other students began working on it using Python and Pygame. I acted as the gameplay programmer as well as the production lead, making sure people were getting work done, staying on task, and staying in line with the project vision. It is currently left in an unfinished state, but for a semesters worth of work, I am quite proud of it. Note that, to play this, you need some IDE that supports Python. There is no executable file, but with the IDE of your choice, simply run the main.py file.

Gravity Penguin 3D

My first big Unreal Engine 4 game, and the second in the Gravity Penguin series of games. This time, it is a 3D speed based platformer based on Crash Bandicoot. I programmed the player gameplay, made the player model, implemented the animations for the player, worked on the level design, and made the post-processing filter to give the game that low-res, PlayStation 1 era kind of feel. Note that you will need a 64-bit system to play using the executable file.

Anomaly

My first Game Jam game that I made with a group of 5 others. This was my first foray into the Unity game engine, and it was a lot of fun to code the player having and learning the engine at the same time. I also implemented all of the animations, and I even made the voxel player model.

Gravity Penguin NES

The third entry in the Gravity Penguin series. This time, it is going back to the 2D perspective and is based on the original Super Mario Bros. for the NES. I'm still currently working on this one, but at the moment, it has a fully playable player character, enemies, a fully functioning hud system, and an in progress tile collision system. I also made all of the art assets.

  • Check it out

Sonic X-Treme Recreation

This is my attempt to recreate the cancelled 3D Sonic game, titled Sonic X-Treme, which was slated to release for the SEGA Saturn in 1996. Using Unity, as well as some of the art assets from the cancelled game leaked to the public, I tried my best to replicate the player movement based on the footage that has been seen of the game. Still a work in progress passion project.
